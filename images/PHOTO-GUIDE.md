## Tips

- **Format:** JPG for photos, PNG only if you need transparency
- **Exact dimensions aren't required** — the CSS crops to fit, so being close to the ratio is what matters most
- **Compression:** Use [Squoosh](https://squoosh.app) to compress before uploading — aim for under 300KB for cards, under 500KB for heroes
- **Naming:** Keep filenames exactly as listed — lowercase, hyphens, no spaces
- **After adding photos:** Push to GitHub with `git add images/ && git commit -m "Add photos" && git push`
